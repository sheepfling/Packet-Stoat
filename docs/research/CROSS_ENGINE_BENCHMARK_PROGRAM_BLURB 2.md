# Cross-Engine Benchmark Program Blurb

FastDIS needs a benchmark program that proves more than parser speed or
one-engine demos. The goal is to run one shared cross-engine harness with the
same DIS traffic scenarios, truth fixtures, and report schema across native,
Python, Unreal, Unity, and Godot, then publish honest same-host head-to-head
comparisons against GRILL wherever GRILL actually installs and runs. When
complete, FastDIS will have one benchmark matrix that links reproducible
ingest, filtering, replay, latest-state, runtime, and cross-engine-equivalence
evidence to every product claim, with explicit boundaries wherever a
competitor comparison is blocked or non-comparable.
