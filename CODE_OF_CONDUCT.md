# Code of Conduct

This project expects direct, respectful, technically grounded collaboration.

## Standards

- discuss facts, tradeoffs, and evidence
- keep disagreement technical and concrete
- do not post secrets, credentials, or private captures
- do not misrepresent mock or experimental integrations as production-verified

## Scope

This codebase includes core parsing, engine workflows, and mock/vendor-adjacent
integration surfaces. Treat claims carefully and keep release language honest.
